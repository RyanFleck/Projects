This ZIP folder contains four folders, labelled Q1 through Q4, which contain 
the Java source code, JavaDoc directories and additional files pertaining to 
each question. Each Q# folder contains a variation of the basic 
MontyHall.java, which is used in conjunction with other classes to print specific information to the console and run N games. Please note that our sample data (CSV type information with and without graphics) is placed in the Q3 folder.
